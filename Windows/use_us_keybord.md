## [Windows環境構築] USキーボードを使用するためにやること

Windows(日本語版)のPCを買ったり組んだりしたとき、いつもやり方忘れて調べるので備忘録として残しておく。

日本語配列のキーボードを使用している場合は不要だが英字（US）配列のキーボードを愛用しているため必須の作業。

### やること
- キーボードのレイアウトを英語キーボードにする
- Ctrl + SpaceでIMEのオンオフができるようにする
- CapsLockをCtrlに ( HHKBの場合は不要 )

### 手順
#### キーボードのレイアウト変更
- 設定 >> 時間と言語 >> 言語( Win10の場合はサイドバーにある ) >> 日本語 と書いてあるところをクリックし、「オプション( 言語オプション )」を開く
- ハードウェア キーボードレイアウトの部分が日本語キーボードになっていると思うので、その部分を英語キーボード( 101/102キー )に変更
#### Ctrl + SpaceでIME切り替えができるように
- 画面右下の「A/あ」の表示を右クリック >> 設定  => IMEの設定画面を開く
- キーとタッチのカスタマイズを選択
- キーの割り当てでCtrl + SpaceをIME オンオフに
#### CapsLock => Ctrl
- Ctrl2Capというツールを使用した
- [参考: 5分でCapsLockキーをCtrlキーに変更する](https://joytas.net/windows/caps2ctrl)
